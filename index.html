Помощь

Добро пожаловать в электронный журнал оценок. Здесь вы можете загружать, просматривать и анализировать оценки студентов. Ниже приведено описание всех вкладок:

📥 Загрузка оценок

Позволяет загрузить файл с оценками в формате .сsv, .txt, .xlsx. После загрузки данные автоматически отобразятся в журнале. Формат CSV: ФИО,Класс,Русский язык,Математика,Физика,Физкультура

📝 Журнал

Здесь отображаются все загруженные данные в виде таблицы. Вы можете:

  • Редактировать оценки вручную, нажав на ячейку.
  • Удалить строку с помощью кнопки "Удалить".
  • Добавить нового ученика вручную через форму ниже таблицы.

📊 Табличная статистика

Отображает сводную таблицу по каждому классу и по всем классам сразу. В таблице представлены:

  • Средняя оценка по предмету
  • Медианная оценка
  • Частота каждой оценки (от 1 до 5)

📈 Графическая статистика

Показывает графики для лучшего визуального анализа:

  • График средней оценки и общего количества оценок по предметам (все классы)
  • Графики средней оценки по предметам для каждого класса
  • Графики распределения оценок по предметам (по классам)
  • Графики количества каждой оценки по каждому предмету
